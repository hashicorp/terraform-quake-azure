# terraform-quake-azure
Example repo to run a Quake client and server on Azure and Azure Container Service with Terraform

## Terraform Config
### Core terraform/core
- remote state
- resource group
- container service cluster

### VM terraform/vm
- Windows VM
- virtual network
- security group
- managed disks
- dns

### K8s Service
- k8s pod
- k8s service
- dns
